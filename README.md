
  # Ecommerce Baby Products

  This is a code bundle for Ecommerce Baby Products. The original project is available at https://www.figma.com/design/3fdYhl5T5FPPvlQkC25I3J/Ecommerce-Baby-Products.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

By default it runs on `http://localhost:6556` (override with `VITE_PORT` or `PORT`).
